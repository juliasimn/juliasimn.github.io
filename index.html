Projects
2020
Pattern Superposition on Digitally Generated Cartoons to Improve Visual Experience for Colorblind users.

In the subjects Tutored Research and Development Projects I and II (in the master), a python application is being implemented to improve the visual experience of coloblind users. For that, a clustering of the colors, for studying which ones are hard to perceive is done, and after that patterns are added in colors that are difficult to difference so color blind users are able to difference them.

2020
Facial Recognition Systems with Convolutional Neural Networks using ethnicity and gender information.

This project was done as a final degree thesis in the computer science degree. It was developed with Python using the CNNs Vgg-16 and Resnet-50.

The objective was to improve the results of those CNNs in face recognition using gender and ethnicity information. For that two databases were used to finetuning the CNNs, and later, another layer based in triplet loss was introduced to improve the results. In the triplet loss the corresponding information of ethnicity and gender were used to retrain the net.

An application in .NET was developed using C# and ASP.NET.

In the application it was possible to record new videos to add to the database, with the Kinect camera, and it was possible to test different heart rate detection algorithms in the different video sequences. To get proper information on how the algorithms were working, in the application, a bluetooth device for heart rate detection was added, that was able to save the real heart rate value while recording the video sequences.
